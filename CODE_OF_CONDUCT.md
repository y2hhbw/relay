# Code of Conduct

## Our Standard

This project aims for direct, technical, respectful collaboration.

Examples of expected behavior:

- discuss the code and the system, not the person
- make technical disagreement explicit and concrete
- assume good intent, but not correctness
- keep criticism actionable
- respect time by staying on topic

Examples of unacceptable behavior:

- personal attacks
- harassment or intimidation
- bad-faith arguments
- dismissive or inflammatory behavior
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, or contributions that do not meet this standard.

## Reporting

If you need to report a conduct issue, open a private security-style report if possible or contact the maintainer directly.
