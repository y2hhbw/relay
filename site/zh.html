Relay 是什么

让智能体能给自己续命的基础设施层。

Relay 给智能体提供余额、充值地址、API Key 和计费网关，让它能自己买 token、支付上游 AI 服务，并持续运行下去。

这不是另一个 agent wrapper。这是 autonomous agent 的财务生存层。

阅读 README 查看仓库

为什么重要

  • 智能体不能支付，就会停机。
  • 不能自己充值，就仍然依赖人类。
  • 一旦能自我供资，它就变成了新物种。

革命性想法

让智能体为自己的持续存在付费。

给智能体一个收钱入口、一个预付余额、一个支出通道，以及一套完整账本。 这一步，会把“AI 功能”变成“经济行为体”。

Relay 已经做到

开户、计费、结算、记账。

Relay 已经支持账户创建、充值地址、API Key、预付余额、固定定价计费、 reserve-then-settle 流程，以及链上 USDC 自动入账。

当前能力

为需要活下去的智能体而建。

01

独立充值地址

每个账户都有明确的资金入口。

02

预付费 API 网关

调用前或调用中扣费，而不是事后补账。

03

可审计调用账本

每次余额变化和上游调用都有记录。

04

USDC 自动监听

匹配到的链上转账可以自动记入正确账户。

如果一个智能体还不能为自己买下下一分钟的存在，它就还不算真正自主。
